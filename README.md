# PimEyes Automation System

## Tech Stack

- Next.js
- FastAPI
- Playwright
- HTTP Toolkit
- Tailwind CSS

## Features

- Image upload automation
- Browser automation
- Search workflow automation
- Result extraction
- Session persistence
- Captcha detection
- Retry logic

## Setup

### Backend

cd backend

pip install -r requirements.txt

playwright install

uvicorn main:app --reload

### Frontend

cd frontend

npm install

npm run dev
